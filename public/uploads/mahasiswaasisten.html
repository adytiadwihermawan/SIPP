SELAMAT DATANG

BERIKUT HALAMAN YANG DAPAT ANDA AKSES

MAHASISWA

    Halaman mahasiswa dapat digunakan mahasiswa untuk mengakses materi praktikum baik dari dosen maupun asistennya.
    Mahasiswa dapat mengisi Absen, mendownload materi dan mengumpulkan tugas melalui halaman ini.

ASISTEN PRAKTIKUM

    Halaman asisten praktikum dapat digunakan mahasiswa yang menjadi asisten praktikum untuk menajamen proses praktikum berdasarkan arahan dari dosen pengampu mata kuliah.
    Asisten praktikum dapat membuat pertemuan, membuat absen, mengupload tugas dan proses-proses terkait praktikum lainnya.

DOSEN

    Halaman dosen dapat digunakan oleh dosen untuk menajamen proses praktikum sesuai kurikulum yang berlaku.
    Dosen dapat membuat pertemuan, membuat absen, mengupload tugas dan proses-proses terkait praktikum lainnya.

ADMIN

    Halaman admin dapat digunakan admin untuk memanajemen kegiatan praktikum.
    Admin dapat membuka kelas praktikum baru, mengatur data user, dan lainnya.
